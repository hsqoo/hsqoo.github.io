EasyMock搭建

easy-mock Mac环境搭建

本地环境：MacOS Catalina 10.15.5 (19F101)

Node.js（v8.x, 不支持 v10.x）& MongoDB（>= v3.4）& Redis（>= v4.0）

一. 安装Mongodb

Mac使用homebrew安装Mongodb

brew install mongodb

创建目录

sudo mkdir -p /data/db

启动mongodb

sudo mongod

二. 安装redis

使用homebrew 安装redis

brew install redis

安装完成后，运行redis

redis-server

三. 安装easy-mock

$ git clone https://github.com/easy-mock/easy-mock.git
$ cd easy-mock && npm install
启动
$ npm run dev
# 访问 http://127.0.0.1:7300
更多
# 前端静态资源构建打包
$ npm run build

# 以生产环境方式启动，需要提前执行 build
$ npm run start

# 单元测试
$ npm run test

# 语法检测
$ npm run lint

运行成功
